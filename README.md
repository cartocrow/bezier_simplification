# Bézier Simplification
This repository is an in-progress of simplification of Bézier curves (aimed at simplifying and smoothing isolines).

## Dependencies
This module depends on the following libraries:

* [Cartocrow/core](https://github.com/cartocrow/core)
* [libspiro](https://github.com/fontforge/libspiro) (is a dependency of ipelib, so may be already installed; vcpkg.json is not yet configured for this)

### Linux

<details>
  <summary><b>Installing dependencies on Linux</b></summary>
